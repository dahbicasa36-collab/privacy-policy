سياسة الخصوصية

نحن نلتزم بحماية خصوصية مستخدمينا وضمان سرية بياناتهم. باستخدامك لتطبيقنا أو متجرنا الإلكتروني، فإنك توافق على بنود هذه السياسة وطرق جمع البيانات ومعالجتها.

1. البيانات التي نقوم بجمعها

  • الاسم الكامل
  • عنوان البريد الإلكتروني
  • رقم الهاتف
  • بيانات الاستخدام مثل الصفحات التي تزورها ومدة الجلسة
  • معلومات تقنية مثل عنوان IP ونوع المتصفح

2. كيفية استخدام البيانات

  • تحسين تجربة المستخدم وتطوير خدماتنا
  • إرسال إشعارات حول المنتجات أو العروض
  • معالجة الطلبات وتسهيل عملية الشراء
  • تقديم الدعم الفني
  • تحليل الأداء وفهم سلوك المستخدم

3. حماية البيانات

نستخدم بروتوكول HTTPS، وتشفير البيانات، وأنظمة حماية متقدمة لمنع الوصول غير المصرح به.

4. مشاركة البيانات

لا نقوم بمشاركة بياناتك مع أي طرف ثالث إلا للضرورات التقنية مثل شركات الشحن أو الدفع أو الامتثال للقانون.

5. حقوق المستخدم

  • طلب الوصول إلى البيانات
  • تعديل أو حذف البيانات
  • سحب الموافقة
  • التواصل معنا لأي استفسار

6. حذف البيانات

يمكنك طلب حذف بياناتك عبر مراسلتنا، وسيتم الحذف بشكل نهائي خلال مدة معقولة.

7. التواصل معنا

للاستفسار حول سياسة الخصوصية، يرجى التواصل عبر البريد أو واتساب الرسمي.
